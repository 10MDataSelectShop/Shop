##  Tech Stack

<details>

<div markdown="1">    
  
### **Application**

- **JAVA 11**
- **Spring Boot** _2.7.0
- **Spring Security** _0.11.2
- **JPA**
- **Query DSL** _5.0.0
- **Full Text Search**

- **AWS RDS - MySQL** _8.028
- **AWS ElastiCache for Redis** _7.0.4
